Startpaket

Für das verbindliche Startpaket, Membercard und eine Gruppeneinweisung, wird eine einmalige Zahlung von 29,99 € fällig; optional kann ein Personaltraining statt der Gruppeneinweisung gebucht werden, die einmalige Zahlung erhöht sich dann von 29,99€ auf einmalig 39,99€.
